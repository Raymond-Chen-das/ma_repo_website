🏃‍♂️ Garmin運動手環多變量分析

探討活動量與睡眠品質間的關聯性

本研究透過系統性的資料處理與探索性分析，深入探討71位Garmin使用者的活動與睡眠模式。 目前已完成嚴格的資料清理（發現重要時區問題）與全面的EDA分析， 建立了「活動-睡眠質量互補假設」的初步證據，為後續多變量統計分析奠定堅實基礎。

🎯 研究概述

研究背景與動機

隨著穿戴式健康裝置的普及，大量的生理與活動資料為健康研究提供了新的機會。本研究專注於探討日常活動量與睡眠品質間的複雜關聯性，透過Garmin運動手環收集的真實世界資料，運用嚴格的資料清理與探索性分析方法進行深度研究。

目前進度：已完成資料清理與探索性分析階段，發現重要的時區問題並建立「活動-睡眠質量互補假設」的初步證據。

系統性資料清理

發現並解決關鍵的時區混亂問題（1.8% → 95.5%正常化），建立嚴格的品質控制流程， 處理69.3%的重複記錄、異常值與日期歸屬邏輯，確保分析結果的學術可信度。

📈 分析流程進度

✅
Phase 1 - 已完成
資料清理與品質控制
重大發現：系統性識別時區問題（1.8% → 95.5%正常化），處理69.3%重複記錄與異常值。 最終從3,976筆原始記錄中提煉出1,031筆高品質配對資料，保留率25.9%。
✅
Phase 2 - 已完成
探索性資料分析 (EDA)
關鍵發現：平均睡眠時間7.47±1.46小時符合WHO標準，但發現活動指標間嚴重多重共線性（r=0.991）。 驗證「活動-睡眠質量互補假設」：高活動量者睡眠時間略短但深睡比例更高。
⏳
Phase 3 - 進行中
多變量統計分析
計劃方法：運用PCA解決共線性問題，K-means聚類識別使用者群體， ANOVA檢驗群體間差異，建立線性迴歸預測模型。
預計將為期末報告提供完整的統計學證據。

🏆 關鍵研究發現（已完成階段）

🌏 時區問題的重大發現
首次系統性識別穿戴式裝置的時區記錄問題：原始資料僅1.8%睡眠記錄出現在正常夜間時段，修正後95.5%恢復正常夜間分布，展現批判性資料檢查的重要性
🔗 活動-睡眠質量互補初步證據
發現活動量與睡眠時長呈弱負相關（r=-0.096），與深睡比例呈弱正相關（r=0.099），支持「高活動量使用者睡眠時間略短但品質更好」的假設
⚠️ 多重共線性挑戰識別
發現步數與活動距離間極高相關性（r=0.991），確認多變量分析中必須進行PCA降維處理，避免統計推論偏誤
📊 久坐生活型態確認
75%使用者日步數低於5,068步，遠低於WHO建議的10,000步，平均睡眠7.47小時符合標準但個體差異顯著（SD=1.46小時）
